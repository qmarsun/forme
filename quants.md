Professional and Feature-Rich Libraries
QuantLib: This is the most comprehensive and well-known open-source C++ library for quantitative finance. It provides a vast collection of models and tools for pricing a wide range of financial instruments, building yield curves, and managing risk.
SS&C Algorithmics Algo Pricing Library Toolkit: A commercial offering from SS&C Technologies that allows seamless integration with advanced proprietary models using C++ or Python APIs.
Bloomberg API Library: While not a complete pricing library itself, it is a crucial API used by financial professionals to access streaming market data and execute trades, which often underpins internal pricing systems built with C++.

Open-Source and Specialized Libraries
These are typically found on GitHub and focus on specific aspects of pricing, often for educational purposes or simpler applications:
FinancialLibrary: A C++ library for financial calculations such as pricing derivatives and building yield curves.
OptionsPricingCPP: High-performance C++ implementation of critical option pricing models, including Black-Scholes, Binomial, Finite Difference, and Monte Carlo methods.
option-pricer: A library designed to model and price various financial options, including European, American, and exotic path-dependent options.
black-scholes-cpp: A comprehensive program focused on the analytic solution of the Black-Scholes equation for European options.

Key Underlying Math Libraries
Quantitative finance often requires complex mathematical operations, and "quants" frequently utilize general-purpose C++ math libraries in conjunction with their pricing code:
Armadillo: A high-quality C++ linear algebra library.
Eigen: Another popular and powerful C++ template library for linear algebra.
Boost.Math: Part of the well-known Boost set of libraries, offering advanced mathematical functions.
GSL: The GNU Scientific Library, a comprehensive numerical library for C and C++ programmers.
